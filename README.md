# ShardKV  
Modern C++ 23 Sharded KV Store  

### Build  
```mkdir build```  
```cd build```  
```cmake .. -G Ninja```  

### Execute  
```./executable```

### Dependancies
OS: ```Linux (WSL)```  
Ready: ```sudo apt update```  
Compiler: ```sudo apt install clang```  
Build Tool: ```sudo apt install cmake```
